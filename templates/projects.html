Will Hue Survive

2016, EPITECH Innovation HUB

Will Hue Survive is a concluded experimentation. We tried to use the Philips Hue as an extension of the game world. If you want to build a game using it, don't expect to have nice thunder effect, it was very random on our project, since the light doesn't isn't reactive fast enough. But if you want to build something like an external radar, it can do it nicely.
For instance, we used them in a demo scene taking place in a car moving forward under street lamps.
